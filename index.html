这个简约的个人网站采用灰白配色方案，完美融合了功能性与美学设计。左侧状态栏采用浅灰色背景，整体页面以灰白为主色调，营造出专业而舒适的氛围。

This minimalist personal website features a gray and white color scheme, perfectly blending functionality and aesthetics. The left sidebar has a light gray background, and the overall page is dominated by gray and white tones, creating a professional and comfortable atmosphere.

网站包含精心设计的元素：启动动画、深色/浅色模式切换、响应式布局以及中英双语支持。所有功能都使用纯HTML、CSS和JavaScript实现。

The website includes carefully designed elements: launch animation, dark/light mode switching, responsive layout, and bilingual support. All features are implemented using pure HTML, CSS, and JavaScript.
